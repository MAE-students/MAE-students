# MSc Music & Acoustic Engineering - Open Knowledge Base

Welcome to the official community hub for the **Music and Acoustic Engineering** Master's degree at **Politecnico di Milano**. This space is designed to archive, preserve, and collectively improve course notes, summaries, and code.

## 🏗️ Organization Structure
To keep materials organized and easy to manage, we use **one repository per subject**.
* Browse the repository list to find the specific exam or course you are looking for.
* All repositories are public and accessible to everyone.

## 🤝 How to Use & Contribute
This project follows the "Open Source" model. Don't just download: if you find an error or have better notes, share them!
Since base permissions are set to **Read**, follow this workflow to contribute:

1. **Fork**: Create a copy of the subject's repository on your own GitHub profile.
2. **Modify**: Add your files or edit existing ones.
3. **Pull Request (PR)**: Submit your changes to the main repository. Admins will review and merge your contribution.

## 📂 Formats & Storage
* **Preferred Formats**: LaTeX (`.tex`), Markdown (`.md`), and Plain Text (`.txt`).
* **Overleaf Integration**: We highly encourage using the Overleaf-GitHub sync for LaTeX projects.
